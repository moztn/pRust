What is this

pRUST is for now a POC of an interactive way to present RUST, the memory safe language, it is meant to evolve into a reveal plugin.

pRUST allow the execution of demonstrated code whitin the presentation.

Find more about Rust

How it work

All you have to do is to use this template, add rust class in any code you want to be executabe.

Editable


fn main() {
    let mut count = 0u32;
    println!("Let's count until infinity!");
    // Infinite loop
    loop {
        count += 1;
        if count == 3 {
            println!("three");
            // Skip the rest of this iteration
            continue;
        }
        println!("{}", count);
        if count == 5 {
            println!("OK, that's enough");
            // Exit this loop
            break;
        }
    }
}
